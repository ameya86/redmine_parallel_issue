= Parallel Issue

同じ内容のチケットを任意の担当者分、同時に作成できます。


== インストール

1. RAILS_ROOT/vendor/pluginsに配置する。

2. Redmineを再起動する。


== アンインストール

1. RAILS_ROOT/vendor/pluginsから削除する。

2. Redmineを再起動する。
